Tetris

scalatetris.Tetris
object Tetris

Objeto principal que implementa la lógica del juego Tetris usando el sistema de actores de Akka.

Este objeto maneja:

  • Los comandos del juego
  • El ciclo de juego
  • La interacción con el motor del juego
  • La gestión del audio
